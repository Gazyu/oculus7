これは何？
------------

Nexus7(2013)をOculus Riftっぽくするアタッチメントです．

詳しくは，C85　ななかINSIDE PRESS vol.4を買ってください．

https://sites.google.com/site/dai7sec/


材料
------

100円ショップで買えます．

色紙： 240x270mm 2枚入り 105円
ルーペ： 倍率5倍  105円 x 4
計： 525円

ルーペは，新宿三丁目のセリアで買ってきた「ポケットルーペ(株式会社グリーンオーナメント)」が良かったです．


あると良い道具
---------------

* カッターナイフ
* デザインナイフ
* コンパス
* 定規・三角定規
* 筆記用具
* 両面テープ

これも足りないものは100円ショップで揃います．
